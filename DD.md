# TeleDot - Design Document

## Scenes
### Main
Have the settings and about buttons on the bottom of the screen with settings right for easier access.

#### Options
- Files panel;
- Settings button;
- About button;

### Settings
For the settings, we should display a preview on top to make the settings easy to notice how they would look in the app.

#### Options
- Speed;
- Font size;
- Enable MarkDown support;
- Margins;
- Enable mirror mode;
- Font color;
- Background color;
- Enable internet (for connecting with pc client);
- Which side controls should be on (left/right);

### Teleprompt
#### Options
- Back button;
- Playback of text itself;
- Remote controller support;
- Manual controlls on side;
